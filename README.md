# Higgs Boson Event Detection using Machine Learning

This project applies Artificial Intelligence and Machine Learning techniques to detect
rare Higgs Boson events from simulated Large Hadron Collider (LHC) collision data.

## Algorithms Used
- Logistic Regression
- Random Forest
- XGBoost
- Neural Network (Feedforward)

## Evaluation Metrics
- Precision
- Recall
- F1-score
- ROC-AUC
- Confusion Matrix

## Dataset
ATLAS Higgs Machine Learning Challenge Dataset (CERN OpenData)

## Environment
- Python
- Jupyter Notebook
- scikit-learn
- XGBoost
- TensorFlow / Keras

This project is part of the CU6051NI Artificial Intelligence module coursework.
